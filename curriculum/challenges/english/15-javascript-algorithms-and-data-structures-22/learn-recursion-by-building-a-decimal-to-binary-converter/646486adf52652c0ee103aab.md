# --description--

Recall that the call stack is a LIFO (last in, first out) data structure. This means that, as functions are called, they are added to the top of the stack, and as functions return, they are removed from the top of the stack.

Treat the `animationData` array as the visual representation of a stack and add a new object to the top of the stack before your first object. Your new object should have the properties `inputVal`, `marginTop`, and `addElDelay` set to `2`, `-200`, and `1500`, respectively.
